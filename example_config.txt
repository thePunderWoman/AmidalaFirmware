# --- Button Mappings ---
# Two mappings are available per button:
#   b=  Short press  — fires on button release (tap)
#   lb= Long press   — fires after holding the button for 3 seconds (3000 ms);
#                      the normal short-press release event is suppressed so only
#                      the long-press action runs.  Both use the same format.
#
# Format: b=ButtonNum,Action,Arg1,Arg2,SerialStr
#         lb=ButtonNum,Action,Arg1,Arg2,SerialStr
#   SerialStr (optional 5th field) sends a serial string side-effect alongside any action.
#
# Actions:
#   1=Sound(bank,file)          file is optional; omit to play random from bank
#   2=Servo(num,pos)
#   3=DigitalOut(num,state)     state: 0=NO, 1=NC, 2=Momentary
#   4=I2CCmd(addr,cmd)
#   5=SerStr(index)             send serial string by 1-based index
#   6=I2CStr(addr,index)
#   7=HCREmote(emotion,level)   emotion: 0=Happy 1=Sad 2=Mad 3=Scared 4=Overload
#                               level:   0=Moderate 1=Strong
#   8=HCRMuseToggle
#   9=DomeCmd(subcmd,angle)     RoboClaw dome drive only (DOME_DRIVE_ROBOCLAW)
#
# DomeCmd subcmds (Action=9):
#   0=RandomToggle   b=N,9,0          toggle random wander mode on/off
#   1=Stop           b=N,9,1          emergency stop
#   2=Front          b=N,9,2          go to front (0°)
#   3=Home           b=N,9,3          re-run homing sweep
#   4=Calibrate      b=N,9,4          run 10-revolution encoder calibration
#   5=GotoAbs        b=N,9,5,<deg>    go to absolute angle (0–255°)
#   6=RelativePos    b=N,9,6,<deg>    rotate +N degrees from current position
#   7=RelativeNeg    b=N,9,7,<deg>    rotate -N degrees from current position
#   8=AbsStickToggle b=N,9,8          toggle absolute-stick mode (joystick angle = dome heading)
#
# DomeCmd examples:
#   b=1,9,0          Button 1: toggle random wander mode
#   b=2,9,1          Button 2: stop dome
#   b=3,9,2          Button 3: go to front
#   b=4,9,5,90       Button 4: go to absolute 90°
#   b=5,9,6,45       Button 5: relative +45°
#   b=6,9,7,45       Button 6: relative -45°
#
# Long-press examples (hold 3 seconds):
#   lb=1,1,3         Long-press button 1: play random sound from bank 3
#   lb=2,7,0,1       Long-press button 2: Very Happy emote
#   lb=3,8           Long-press button 3: toggle HCR Muse
#   lb=4,9,3         Long-press button 4: re-run dome homing sweep
#   lb=5,9,4         Long-press button 5: run dome encoder calibration
#
# ---- Alt button (modifier layer) ----
# altbtn=N  Designates button N as a hold-to-activate modifier (default 0 = disabled).
#   While held, all other buttons dispatch their alt-layer action (ab=) instead of
#   their normal action (b=).  Normal long-press actions (lb=) are suppressed while
#   alt is held.  The alt button itself never fires its own b= or lb= action.
#
# Button numbering: drive-stick triangle=1, circle=2, cross=3, square=4, l3=5
#                   dome-stick  triangle=6, circle=7, cross=8, square=9
#   (Dome l3 is reserved for gesture input and cannot be the alt button.)
#
# altdomestick=N  What the dome stick does while alt is held (RoboClaw only).
#   0 = no change (default)   dome stick behaves normally
#   1 = abs-stick mode        stick angle maps directly to a dome heading
#
# ab=  Alt-button action — same format as b=.  Only fires when altbtn is held.
#
# Alt-button examples:
#   altbtn=5         Drive l3 is the alt modifier
#   altdomestick=1   Dome stick enters abs-stick mode while alt is held
#   ab=1,5,17        Alt+button 1: send serial string #17
#   ab=2,7,0,1       Alt+button 2: Very Happy emote
#   ab=3,9,0         Alt+button 3: toggle random dome mode
#   ab=4,9,3         Alt+button 4: re-run dome homing sweep
#
# Gestures use the same action/subcmd numbers but drop the button number:
#   g=GestureStr,9,0          gesture triggers random toggle
#   g=GestureStr,9,5,90       gesture goes to absolute 90°
